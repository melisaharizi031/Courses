#Training-Center
This project was created as part of my hands-on practice while learning web design. The primary goal was to design and build a functional, user-friendly and visually clean website for a training course.

#Project description
This website showcases a training center, offering details about avaliable courses. The desing is clean and minimalist, emphasizing clear content structure and an intuiyive user experience.

#Programming languages
-Html - used to structure page
-CSS - used to style page
-JavaScript - used on the client side to handle form submission
-PHP - used on the server side to proces form data, alide input, connect to the database
-MySQL - used as database to store application data such as name, email and selected course.
